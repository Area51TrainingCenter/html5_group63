Este es una ejemplo de las listas

lo primero que debemos de saber es que existen 2 tipos de listas

Listas ORDENADAS

este tipo de lista se utilizan cuando la secuencia de los sucesos importa

  1. Levantarme 530 am
  2. hacer ejercicios
  3. tomar una ducha
  4. alistarme al trabajo

Listas DESORDENADAS

este tipo de lista no importa la secuencia

  • Rojo
  • Azul
  • Verde
  • Amarrillo
